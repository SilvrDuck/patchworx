# z3

A zig s3 clone heavily inspired by MinIO.

## Quickstart

Needs ZIG 0.13.0

```
zig build
zig-out/bin/z3
```

## API

Uses ranges https://www.rfc-editor.org/rfc/rfc9110.html#rule.ranges-specifier

## Architecture

Uses the hexagonal architecture.
